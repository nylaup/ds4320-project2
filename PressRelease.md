# Will it flip? Predicting county vote flips in the next presidential election

## Using demographic changes and results from the 2020 election to predict county flips for future elections

## Problem Statement        
Currently, there are many different methods used to predict election outcomes, many centering around forecasters who draw on their own experiences and trends they have been observing in order to state what they think the outcome will be, which have varying levels of success. Not many election predictions focus on voter flips in elections, which are often very integral to the overall result.       

## Solution Description      
In order to hopefully generate a useful tool for predicting elections, I decided to use a Logistic Regression machine learning model to predict if a county was going to flip or not. This model uses presidential election results, as well as other information on demographic and economic changes, to learn how to be able to predict which counties' votes flipped in the 2020 election. It will then apply this learned relationship to given information about counties leading up to the 2024 election to then predict which counties will flip from the way they voted in the last presidential election.
